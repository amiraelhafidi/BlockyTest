Testen met blokken

Testautomatisering zonder code, ook voor beginners.

BlockyTest is gemaakt voor studenten, testers en teams die tests willen bouwen zonder veel programmeerervaring. Je stelt testscenario's samen met blokken, voert ze uit en bekijkt direct het resultaat.

Hoe werkt BlockyTest?

Bouw 1. Bouw je test Kies blokken zoals browser openen, wachten en sluiten.
Code 2. Genereer code BlockyTest zet je blokken om naar testcode.
Run 3. Run en bekijk resultaat Voer de test uit en zie of hij geslaagd of gefaald is.
